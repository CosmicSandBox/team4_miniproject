## "들어봤니 모현의 숲🏞" 은 
- 누구나 참여할 수 있는 재미있는 참여형 테스트입니다.

### 기획 배경
- 글로벌 캠퍼스의 “모현의 숲" 별명에서 착안해 외대인 특색을 살린 동물 테스트
- 비교적 시골에 위치해 산 속에 둘러쌓인 글캠에 산동물이 자주 출연하는데서 고안
- 외대 글캠생들의 정체성 확립 + 자존감 지킴이 + 캠퍼스 홍보효과를 위해 기획됨
